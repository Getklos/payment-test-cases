# Ручные тест-кейсы для проверки главной страницы Playwright

## TC-PLAY-001 Проверка заголовка в Chrome

**Предусловия:**  
- Установлен Google Chrome (последняя версия)

**Шаги:**
1. Открыть Chrome
2. Перейти по ссылке: https://playwright.dev/
3. Дождаться загрузки страницы

**Ожидаемый результат:**  
Заголовок вкладки содержит слово **Playwright**  


**Окружение:** Chrome + Windows
