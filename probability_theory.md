# TC-001
Проверка правильности расчёта вероятности выпадения ровно двух орлов при трёх подбрасываниях монеты
## Шаги воспроизведения (проверка расчёта вручную)

1. Перечислить **все возможные исходы** трёх подбрасываний монеты  
   (О — орёл, Р — решка)

   Ожидаемый список (должен быть ровно 8 комбинаций):
   1. О О О
   2. О О Р
   3. О Р О
   4. Р О О
   5. О Р Р
   6. Р О Р
   7. Р Р О
   8. Р Р Р

3. Подсчитать общее количество возможных исходов  
   → Ожидаемый результат: **8**

4. Выделить комбинации, в которых **ровно 2 орла**  
   Ожидаемый список:
   - О О Р
   - О Р О
   - Р О О

   → Ожидаемый результат: **3 благоприятных исхода**

5. Рассчитать классическую вероятность  
   P(ровно 2 орла) = количество благоприятных исходов / общее количество исходов  
   → Ожидаемый расчёт: 3 / 8

 ## Ожидаемый результат
- Вероятность выпадения **ровно двух орлов** при трёх подбрасываниях честной монеты равна **3/8**  
- В десятичном виде ≈ **0.375** (37,5 %)  
- Дробь **3/8** не сокращается дальше
